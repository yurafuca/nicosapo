nicotap のご紹介

nicotap はニコニコ動画と生放送を視聴できるアプリです．バックグラウンド再生とポップアップ再生に対応しており，Android Lolipop 以上で動作します．無料で利用できます．

よろしければリツイートしてあげてください．

nicotap というニコニコ動画・生放送用のアプリを作りました．ポップアップ再生とバックグラウンド再生に対応しています．数秒単位で動画をスキップできる機能や音楽プレイヤー機能など，かゆいところに手が届く機能をたくさん実装しています．興味があるかたはぜひどうぞ ;) https://t.co/q8mwSWbJrJ pic.twitter.com/N85CGRl6jF

バグを発見した・機能要望がある

にこさぽのバグレポートの対応や機能要望の対応をしているととてもつらい思いをすることが多いです．
個人開発者は仕事が終わったあとの時間や通勤電車，休日などを使ってアプリやサービスを開発しています．
優しくしてください．

基本的な姿勢として下記の点にかならず留意してください．

  • バグは直ったらラッキーだと思ってください．
  • 要望は受け入れられたらラッキーだと思ってください．
  • バグを効率的に報告・退治して問題を解決するための手助けをしてください．
  • 開発者とユーザは対等だと思っています．優しく報告してください．
  • 報告は作業の完了ではありません．自身の報告にオーナーシップをもってください．
  • 横柄なバグレポートや機能要望が届くと悲しいですし，ひどいものは見なかったことにしています．

レポートおよび要望は下記で受けつけています．管理が大変なのでできる限り GitHub Issues に送ってくださると嬉しいです．非エンジニアのかたに限りお題箱でも受けつけています．

  • エンジニア向け: Github Issues
  • 非エンジニア向け: お題箱

バグレポートを送る場合はバグの修正をただしく助けるレポートを書いてください．たとえば，下記はかならず示してください．下記が示されていないレポートはお受けしません．
